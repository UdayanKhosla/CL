# CommerceLab

A tool-first Commerce platform for CBSE Class XI-XII students.

## Current prototype

- Interactive Class XI market-equilibrium graph
- Seven demand and supply scenarios
- Directly draggable demand and supply curves
- Live equilibrium price and quantity calculation with integer-only display
- Dynamic explanation
- Five-round Class XII Accountancy Ratio Practice challenge
- Clickable financial-statement figures, working validation and interpretation
- 50-card Class XII Cash Flow Classification sprint
- Operating, investing and financing mastery with instant rule explanations
- Mistake review, streak feedback and equal category weighting
- Class XII National Income calculator with income, expenditure and value added methods
- Interactive Class XII Management Function Flow
- NCERT-aligned meaning, features, importance and process for all five functions
- Separate Dashboard, Tools, tool, Sources and About pages
- Primary-source library and dated research notes
- Open access with no login, profile or personalized progress tracking
- Tool directories for Accountancy, Business Studies and Economics

## Run locally

No package installation is required.

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Academic note

In the current CBSE 2026-27 curriculum, market equilibrium is Class XI Introductory Microeconomics. National Income and Related Aggregates and the five management functions are Class XII topics.

See [research/SOURCES.md](research/SOURCES.md) for the source hierarchy and review notes.
